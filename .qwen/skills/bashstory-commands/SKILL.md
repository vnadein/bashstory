# BashStory Commands Skill

## Назначение
Разработка команд терминала для проекта BashStory.

## Контекст проекта
- **Фреймворк:** Next.js 15+ с App Router
- **Язык:** TypeScript
- **Команды:** Находятся в `lib/commands/`
- **API:** `/api/command` endpoint

## Структура команд
```
lib/commands/
├── index.ts          # Реестр команд
├── types.ts          # Типы команд
└── impl/             # Реализации команд
    ├── help.ts
    ├── whoami.ts
    └── ...
```

## Примеры использования
- Создание новых команд терминала
- Добавление алиасов
- Модификация существующих команд
- Тестирование команд

## Связанные MCP серверы
- `filesystem` - чтение/запись файлов команд
- `sqlite` - доступ к данным для команд
- `terminal` - тестирование команд
