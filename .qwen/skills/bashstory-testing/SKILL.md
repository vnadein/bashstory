# BashStory Testing Skill

## Назначение
Тестирование и отладка проекта BashStory.

## Контекст проекта
- **Фреймворк:** Next.js 15+
- **Язык:** TypeScript
- **БД:** SQLite

## Команды для тестирования
```bash
# Запуск разработки
pnpm dev

# Сборка
pnpm build

# Линтинг
pnpm lint

# Проверка TypeScript
npx tsc --noEmit

# Прямой доступ к БД
sqlite3 data/quotes.db
```

## Проверка функциональности
1. **Команды терминала:**
   - `help` - список команд
   - `whoami` - текущий пользователь
   - `posts` - список постов
   - `timeline` - лента

2. **БД:**
   - Проверка подключения
   - Чтение/запись данных

3. **UI:**
   - Отображение терминала
   - Анимации
   - Ввод команд

## Связанные MCP серверы
- `terminal` - запуск команд тестирования
- `sqlite` - проверка данных
- `puppeteer` - браузерные тесты
- `git` - коммит изменений
