# 🚀 Quick Start: MCP + Skills для BAJOUR Social

## Настройка выполнена ✅

### MCP-серверы (8 шт.)
Находятся в `~/.qwen/mcp_servers.json`:

| Сервер | Назначение |
|--------|------------|
| **filesystem** | Чтение/запись файлов проекта |
| **sqlite** | Прямой доступ к БД SQLite |
| **git** | Управление Git (коммиты, ветки) |
| **github** | GitHub API (PR, issues) |
| **memory** | Долговременная память |
| **brave-search** | Поиск документации |
| **puppeteer** | Браузерные тесты |
| **terminal** | Выполнение команд |

### Skills (4 шт.)
Находятся в `~/.qwen/skills/`:

| Skill | Для чего |
|-------|----------|
| **bashstory-commands** | Создание команд терминала |
| **bashstory-db** | Работа с базой данных |
| **bashstory-ui** | UI компоненты терминала |
| **bashstory-testing** | Тестирование и отладка |

## Переменные окружения (опционально)

Для полной функциональности добавьте в `~/.zshrc`:

```bash
# GitHub API
export GITHUB_TOKEN="ghp_..."

# Brave Search API
export BRAVE_API_KEY="..."
```

После добавления:
```bash
source ~/.zshrc
```
